# Open by Design

This self-hosted path is intended to be used, studied, adapted, independently
operated, and forked.

## Code, Releases, and Technical Materials

Code, schemas, scripts, release tooling, and install materials in this
repository are available under the Apache License 2.0. You may use, modify,
redistribute, and include them in commercial or non-commercial work under that
licence.

Independent self-hosted deployments, compatible services, applications, agents,
and forks are welcome. No hosted provider has an exclusive right to implement
PIOS.

## Framework Documentation

The broader PIOS 2.0 framework documentation is published separately under
CC BY 4.0 in the [PIOS repository](https://github.com/peecos/pios). For adapted
framework material, use the attribution guidance in that repository.

## Names, Logos, and Endorsement

You may accurately say that an independent project is based on PIOS,
implements PIOS concepts, or is PIOS-compatible when that is true. Do not claim
to be the official PIOS initiative or to be officially endorsed without
permission.

This boundary protects clarity and trust, not control over independent work.
See [TRADEMARKS.md](TRADEMARKS.md).
